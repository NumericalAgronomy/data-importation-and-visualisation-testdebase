This project aims to perform a comprehensive analysis of the spectral dataset, including the associated species. 

## Context

- **Required Expertise :**  
  The objective is to leverage expertise in AI and data science to carry out an in-depth analysis of the dataset.  
- **Project Phase :**  
  This analysis takes place in the initial stage of model development, enabling a better understanding and exploration of the richness of the data.

## Objectives of the Python Script or Python Notebook

The Python script to be generated should:
- Analyze the entire dataset.
- Use commonly applied analysis methods in the field to extract all relevant information.
- Produce various plots and visualizations to highlight the key characteristics of the dataset:
  - Visualization of raw data.
  - Computation and representation of derivatives.
  - Plotting of relevant statistical and exploratory analyses.
  
The analysis should cover all aspects in order to properly prepare the dataset for the next stages of predictive model development.

## Key Points to Address

- **Data Loading and Preparation :**  
  Implementation of the data loading pipeline and preparation for analysis.
- **Exploratory Statistical Analysis :**  
  Computation and visualization of descriptive statistics and distributions.
- **Visualizations :**  
  - Plotting of spectral curves.
  - Representation of derivatives to better understand variations.
  - Other relevant graphs to detect potential correlations or clusters.
- **Documentation and Explanations :**  
  Inclusion of detailed comments and explanations in the code to facilitate understanding and reuse of the script.
